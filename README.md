# Dexter Benson - Personal Website

This repository contains the source code for my personal website. The site serves to showcase my professional skills, projects, and technical blog posts.

[**View Live Website**](https://dexterbenson.com/)

---

## Technologies

- **Frontend** HTML, CSS, TypeScript, React, Next.js
- **Styling** Tailwind CSS
- **Hosting** Vercel

## Running the Project Locally

### Prerequisites

Install the following software:

- [Node.js](https://nodejs.org)
- [Git](https://git-scm.com)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/dexben5/personal-website
```

2. Navigate into the project directory:

```bash
cd personal-website
```

3. Install dependencies

```bash
npm install
```

4. Start local development server:

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:3000` to view the running project.

---

## Deployment

This website is automatically built and deployed via integration with **Vercel** whenever changes are pushed to the `main` branch. A preview deployment is created when a branch other than `main` has changes pushed to it.
